# 视频标注采集工具 📹

一个用于实时视频拍摄过程中记录观看者反馈标注的Web工具。

## 功能特点

### 🎯 核心功能

1. **实时计时器**
   - 精确的毫秒级计时（格式：MM:SS.mmm）
   - 支持开始、暂停、结束操作
   - 暂停后可继续，时间自动累加

2. **快速标注**
   - 9个预设标注选项：快点、慢点、轻点、重点、舒适、难过、紧张、兴奋、平静
   - 自定义文字输入
   - 一键记录当前时间戳

3. **标注管理**
   - 实时显示所有标注列表
   - 按时间顺序自动排序
   - 支持删除单条标注
   - 统计预设和自定义标注数量

4. **数据持久化**
   - 自动保存到浏览器 localStorage
   - 刷新页面不会丢失数据
   - 支持导出 JSON 格式到本地

## 使用方法

### 1. 打开工具

在浏览器中打开 `video-annotation-tool.html` 文件。

### 2. 开始录制

1. 点击 **"▶ 开始"** 按钮开始计时
2. 计时器开始运行，状态显示为 "● 正在录制"
3. 此时可以添加标注

### 3. 添加标注

**方式一：使用预设选项**
- 点击任意预设按钮（如"快点"、"舒适"等）
- 系统自动记录当前时间戳

**方式二：自定义输入**
- 在输入框中输入自定义内容
- 点击 **"添加"** 按钮或按回车键
- 系统自动记录当前时间戳

### 4. 暂停和继续

- 点击 **"⏸ 暂停"** 按钮暂停计时
- 点击 **"▶ 继续"** 按钮恢复计时
- 暂停期间无法添加标注

### 5. 结束录制

- 点击 **"⏹ 结束"** 按钮停止计时
- 状态变为 "已结束"
- 可以查看和导出数据

### 6. 导出数据

- 点击 **"💾 导出 JSON"** 按钮
- 文件自动下载，命名格式：`annotations_YYYYMMDD_HHMMSS.json`

### 7. 清空数据

- 点击 **"🗑️ 清空数据"** 按钮
- 确认后清空所有标注和计时数据
- **注意：此操作不可恢复！**

## 快捷键

- **空格键**：开始/暂停计时（输入框外）

## 导出数据格式

导出的 JSON 文件包含以下信息：

```json
{
  "recording_name": "录制_2025/11/4 10:30:00",
  "total_duration": 123.456,
  "total_duration_formatted": "02:03.456",
  "annotations": [
    {
      "timestamp": 12.345,
      "time_formatted": "00:12.345",
      "label": "快点",
      "type": "preset"
    },
    {
      "timestamp": 25.678,
      "time_formatted": "00:25.678",
      "label": "自定义内容",
      "type": "custom"
    }
  ],
  "created_at": "2025-11-04T02:30:00.000Z",
  "session_id": 1730678400000
}
```

### 字段说明

- `recording_name`: 录制会话名称
- `total_duration`: 总时长（秒）
- `total_duration_formatted`: 格式化的总时长
- `annotations`: 标注数组
  - `timestamp`: 时间戳（秒）
  - `time_formatted`: 格式化的时间
  - `label`: 标注内容
  - `type`: 标注类型（preset=预设，custom=自定义）
- `created_at`: 导出时间（ISO 8601格式）
- `session_id`: 会话唯一标识

## 使用场景

本工具适用于以下场景：

1. **视频内容测试**
   - 记录观看者在观看视频时的即时反馈
   - 标记特定时间点的情绪或动作反应

2. **用户体验研究**
   - 收集用户在体验过程中的实时反馈
   - 分析特定时间点的用户行为

3. **数据采集**
   - 为视频内容添加时间戳标签
   - 创建训练数据集

## 注意事项

1. **数据保存**
   - 数据自动保存在浏览器 localStorage 中
   - 清除浏览器数据会丢失未导出的标注
   - 建议定期导出重要数据

2. **最佳实践**
   - 拍摄前先测试工具是否正常
   - 重要录制建议实时导出数据备份
   - 避免在计时过程中刷新页面

3. **性能**
   - 工具已优化，可处理大量标注
   - 建议每次录制后及时导出和清空数据

## 故障排除

### 问题：刷新后数据丢失
- 检查浏览器是否启用 localStorage
- 确认没有使用隐私/无痕模式

### 问题：导出按钮不可用
- 确保至少添加了一条标注
- 检查浏览器是否允许下载文件

### 问题：标注按钮无法点击
- 确保已点击"开始"按钮
- 检查状态是否显示"正在录制"
